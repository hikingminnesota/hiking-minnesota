# Hiking Minnesota

**Hiking Minnesota** is a React.js hiking fan site featuring  
trail information, region summaries, and fifty customized badges to earn. 

The live site is here:  
https://hikingminnesota.com/

## Features
- 50 unique and interesting hiking badges  
- Badge earning through optional geolocation  
- Trail descriptions across all major Minnesota regions  
- North Shore, Central, Southeast, Southwest, and more  
- Hosted on GitHub Pages with a custom domain
